Header

  • * 1200 grid
  • * PC는 left 정렬 Mobile은 center 정렬을 기본으로 하되, 프로젝트 예외를 허용한다.
  • * 콜라보 로고가 있을 경우와 없을 경우 모두 대응하도록 한다.
  • * 콜라보 로고와 KP보험서비스 로고를 분리하여 관리 할 수 있도록 한다.
  • * 한개의 어드민 등록 로고로 PC와 Mobile 둘 다 대응할 수 있도록 한다.
       PC - height: 100px;
       Mobile - height: 65px; padding-top: 10px;
  • * Mobile 로고의 max-width는 300px이어야 한다.(콜라보 로고 포함)
